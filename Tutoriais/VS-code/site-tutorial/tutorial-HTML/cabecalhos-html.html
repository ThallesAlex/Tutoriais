Cabeçalhos HTML

Os cabeçalhos HTML são definidos com as tags de <h1> até <h6>

<h1> define o título mais importante, <h6> define o menos importante

Exemplo

<h1>Isso é um título h1</h1>

<h2>Isso é um título h2</h2>

<h3>Isso é um título h3</h3>

<h4>Isso é um título h4</h4>

<h5>Isso é um título h5</h5>

<h6>Isso é um título h6</h6>

Nota: Os navegadores adicionam automaticamente algum espaço em branco (uma margem) antes e depois de um título.

Os títulos são importantes

Os mecanismos de pesquisa usam os títulos para indexar a estrutura e o conteúdo de suas páginas da Web

Ow usuários geralmente folheiam uma página por seus títulos. É importante uasr os títulos para mostrar a estrutura do documento.

Os títulos <h1> devem ser usados pelo títulos principais, seguidos pelos títulos <h2> e <h3>, e assim por diante.

Nota: Use cabeçalhos HTML apenas para títulos. Não use títulos para torna o texto GRANDE ou em

negrito

Títulos maiores

Cada cabeçalho HTML tem um tamanho padrão. No entanto, você pode especificar o tamanho de qualquer título com o atributostyle, usando a propriedade CSS: font-size

Exemplo

<h1 style="font-size:60px">Isso é um título h1</h1>

<h2 style="font-size:50px">Isso é um título h2</h2>

<h3 style="font-size:40px">Isso é um título h3</h3>

<h4 style"font-size:30px">Isso é um título h4</h4>

<h5 style"font-size:20px">Isso é um título h5</h5>

<h6 style"font-size:10px">Isso é um título h6</h6>
