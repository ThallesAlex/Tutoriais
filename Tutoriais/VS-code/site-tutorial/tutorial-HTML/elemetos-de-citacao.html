Nesse capítulo abordaremos o <q>,<blockquote>,<abbr>,<address>,<cite>,<bdo> e o <dfn>.

Exemplo

HTML <blockquote> para Citações

O elemento HTML <blockquote> define uma seção que é citada de outra fonte.

O conteúdo dentro do elemento <blockquote> é geralmente exibido como um bloco separado, e o navegador geralmente adiciona uma indentação (margem) antes e depois do bloco.

Exemplo

<blockquote>Durante 60 anos, o WWF trabalhou para ajudar as pessoas e a natureza a prosperar. Como principal organização conservacionista do mundo, o WWF trabalha em quase 100 países. Em todos os níveis, colaboramos com pessoas ao redor do mundo para desenvolver e fornecer soluções inovadoras que protejam comunidades, a vida selvagem e os lugares em que vivem.</blockquote>

HTML <q> para citações curtas

O elemento HTML <q> define uma citação curta.

Os navegadores geralmente exibem o conteúdo do elemento <q> entre aspas.

Exemplo

Ele disse <q>O tempo é dinheiro</q> e saiu.

Saída

Ele disse "O tempo é dinheiro" e saiu.

HTML <abbr> para abreviações e acrônimos

O elemento HTML <abbr> define uma abreviação ou acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", etc.

Marcar abreviações pode fornecer informações úteis para navegadores, mecanismos de pesquisa e outras tecnologias, como leitores de tela.

Dica: Use o atributo title para fornecer o significado completo da abreviação ou acrônimo.

Exemplo

O <abbr title="World Health Organization">WHO</abbr> foi fundado em 1948.

Saída

O WHO foi fundado em 1948.

PontoUse o atributo title para fornecer o significado completo da abreviação ou acrônimo.

HTML <address> para informações de contato

A tag HTML <address> define informações de contato para o autor/ proprietário de um documento ou artigo.

As informações de contato podem incluir um endereço físico, um endereço de e-mail, um número de telefone, um URL, etc.

O conteúdo dentro do elemento <address> é geralmente exibido em itálico pelo navegador.

Sempre adicione uma quebra de linha (<br>) entre cada item de informação.

Exemplo

<address>
Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA<address>

HTML <cite> para títulos do trabalho

A tag HTML <cite> define o título de um trabalho, como um livro, um poema, um artigo, etc.

Nota:O nome do autor não deve ser incluído no elemento <cite>, a menos que o nome do autor faça parte do título do trabalho.

O conteúdo dentro do elemento <cite> é geralmente exibido em itálico pelo navegador.

Exemplo

Meu livro favorito é <cite>The Scream</cite> de Edvard Munch.

HTML <bdo> para substituir a direção do texto

O elemento HTML <bdo> (bidirectional override) substitui a direção do texto padrão.

O atributo HTML dir especifica a direção do texto.

O atributo dir pode ter dois valores:

  • ltr: da esquerda para a direita (padrão)
  • rtl: da direita para a esquerda

Saída

Exemplo

<bdo dir="rtl">Este texto é exibido da direita para a esquerda.</bdo>
