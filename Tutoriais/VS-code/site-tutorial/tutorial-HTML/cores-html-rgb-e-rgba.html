Cores HTML - RGB e RGBA

Um valor de cor RGB representa a luz VERMELHA, VERDE e AZUL Fontes.

p>Um valor de cor RGBA é o mesmo que RGB, mas com um canal alfa - que especifica a opacidade da cor.

Valores de Cor RGB

Um valor de cor RGB é especificado com a função rgb(), que aceita três valores inteiros entre 0 e 255, representando a intensidade das cores VERMELHA, VERDE e AZUL.

isso significa que existem 256 tons de cada cor, e mais de 16 milhões de combinações possíveis.

para exibir a cor verde, você usaria o seguinte código:

Cor de fundo verde (RGB rgb(0, 255, 0)

para exibir a cor vermelha, você usaria o seguinte código:

Cor de fundo vermelha (RGB rgb(255, 0, 0)

para exibir a cor azul, você usaria o seguinte código:

Cor de fundo azul (RGB rgb(0, 0, 255)

para exibir a cor preto, você usaria o seguinte código:

Cor de fundo preto (RGB rgb(0, 0, 0)

para exibir a cor branco, você usaria o seguinte código:

Cor de fundo branco (RGB rgb(255, 255, 255)

Tons de cinza

Os tons de cinza são criados quando os valores de vermelho, verde e azul são iguais. Por exemplo, rgb(0, 0, 0) é preto, rgb(255, 255, 255) é branco e rgb(128, 128, 128) é cinza médio.

<p style="background-color: rgb(50, 50, 50); color: white;"Cor de fundo cinza(RGB rgb(50, 50, 60)

<p style="background-color: rgb(100, 100, 100); color: white"Cor de fundo cinza(RGB rgb(100, 100, 100)

<p style="background-color: rgb(128, 128, 128); color: white color: white;"Cor de fundo cinza médio (RGB rgb(128, 128, 128)

Valores de Cor RGBA

Os valores de cor RGBA são uma extensão dos valores de cor RGB com um canal alfa - que especifica a opacidade de uma cor.

Um valor de cor RGBA é especificado com:

rgba(red, green, blue, alpha)

onde o valor alfa é um número entre 0.0 (totalmente transparente) e 1.0 (totalmente opaco).

<p style="background-color: rgba(0, 255, 0, 0.5);"Cor de fundo verde (RGBA rgba(0, 255, 0, 0.5)

<p style="background-color: rgba(255, 0, 0, 0.5);"Cor de fundo vermelho (RGBA rgba(255, 0, 0, 0.5)

<p style="background-color: rgba(0, 0, 255, 0.5);"Cor de fundo azul (RGBA rgba(0, 0, 255, 0.5)

<p style="background-color: rgba(0, 0, 0, 0.5); color: white;"Cor de fundo preto (RGBA rgba(0, 0, 0, 0.5)

<p style"background-color: rgba(255, 255, 255, 0.5);"Cor de fundo branco (RGBA rgba(255, 255, 255, 0.5)
