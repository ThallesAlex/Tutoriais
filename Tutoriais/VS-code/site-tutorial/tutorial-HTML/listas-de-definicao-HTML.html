Listas de Definiçãõ (<dl>, <dt>, <dd>)

As listas de definição apresentam pares termo/descrição, ideis para glossários, dicionários, FAQs, ou sempre que for necessário assosiar termos a suas respectivas descrições. Ao contrário das listas <ul> ou <ol>, não há marcadores automáticos ou numeração visual padrão

Estrutura Básica

HTML


<dl>
   <dt>HTML</dt>
   <dd>Linguagem de marcação usado pra criação usada na criação de páginas Web
   <dt>CSS</dt>
   <dd>Folhas de estilo para definir aparência de páginas
</dl>
Saída
HTML
Linguagem de marcação usado pra criação de páginas Web
CSS
folhas de esilo para definir aparência de páginas
  • <dl>inicia a lista de definição.
  • <dt>define o termo.
  • <dd>apresenta a descrição daquele termo.

Exemplo prático: Glossário simples

HTML


<dl>
   <dt>Loop</dt>
   <dd>Estrutura de repetição em programação</dd>
   <dt>Var</dt>
   <dd>Espaço para armazenar valores temporários</dd>
</dl>

Exemplo: Lista de propriedades(metadados)

HTML


<dl>
   <dt>Nome</dt>
   <dd>João Silva</dd>
   <dt>Email</dt>
   <dd>joao@email.com</dd>
</dl>

É possível ter vários <dd> seguidoas por um mesmo <dt>, e vice-versa, comforme a estrutura desejada

Dicas semânticas e limitações

  • Use listas de definição quando o relacionamento termo/descrição fizer sentido
  • Não use <dl> apenas para recuar ou formatar textos; mantenha seu uso para informação estruturada e semântica.
  • Diferente de <ul> ou <ol>, as listas de definição não têm atributos de ordem ou tipo de macador
  • Para fins visuais, utilize CSS para controlar espaçamento, fonte, e layout dos pares termo/definição
