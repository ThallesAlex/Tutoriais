Imagens HTML

As imagens podem melhorar o design e a aparência de uma página da web.

Exemplo

<img src="/VS-code/HTML/imagens.html/O penitente.jpeg" alt="penitente silencioso">

<img src="/VS-code/HTML/imagens.html/purple swirling star wallpaper.jpg" alt"purple swirling star wallpaper">

Sintaxe de imagens HTML

A tag HTML <img> é usada para incorporar uma imagem em uma página da web.

As imagens não são tecnicamente inseridas em uma página da web; As imagens estão vinculadas à páginas web. A tag <img> cria uma retenção espaço para a imagem referênciada.

A tag <img> está vazia, contém apenas atributos e não tem tag de fechamento.

A tag <img> tem dois atributos:

  • src - Especifica o caminho para a imagem
  • alt - Especifica um caminho altenativo para o imagem

Sintaxe

<img src="url" alt"texto alternativo">

O atributo src

O atributo src especifica o caminho (URL) para a imagem que você deseja exibir.

Nota: O caminho da imagem é obrigatório. Sem ele, a tag <img> não exibirá nada.

O caminho da imagem pode ser um URL absoluto ou um URL relativo:

  • URL absoluto - Um URL completo para uma imagem (inclui o protocolo, como http: // ou https: //)
  • URL relativo - O caminho para uma imagem em relação à página da web atual

Nota: Quando uma página da web é carregada, o navegador faz uma solicitação para o servidor para obter a imagem especificada no atributo src. Se a imagem não puder ser encontrada, o navegador exibirá uma imagem de substituição ou um espaço em branco.Os visitantes receberão um ícone de imagem quebrada.

Tipo isso:

O atributo alt

O atributo requerido alt fornece um texto alternativo para a imagem, se o usuário por algum motivo não puder visualizar a imagem(devido a uma conexão lenta, um erro no src ou se o usuário usa um leitor de tela).

O que estiver dentro do alt deve descrever a imagem:

Exemplo

<img src="/VS-code/HTML/imagens.html/8d72c184-d311-4436-ad64-28a126b36e98.jpeg" altGuerreiro com sua espada em uma caverna

Se o navegador não conseguir encontrar uma imagem, ele exibirá o valor do atributo alt:

Exemplo

<img src="eu sei lá" alt="Guerreiro com sua espada em uma caverna">

Tamanho da imagem - Largura e Altura

Você pode usar o atributo style para especificar a largura e altura de uma imagem

Exemplo

<img src="sei lá" alt="Guerreiro com sua espada em uma caverna" style="width: 500px; height: 600px;">

Como alternativa, você pode usar os atributos width e height:

Exemplo

<img src="sei lá.jpg" alt="uma imagem qualquer" width="500px" height="600px"

Os atributos width e height sempre definem a largura e a altura da imagem em pixels.

Nota: Sempre especifique a largura e a altura de uma imagem. Se a largura e a altura não forem definidas, a página web pode piscar enquanto a imagem é carregada.

Largura e Altura, ou Estilo?

Os atributos width ,height e style, são todos válidos em HTML

O atributo style tem a vantagem de poder definir a largura e a altura em unidades diferentes (%, em, rem, etc.) e pode definir outras propriedades CSS, como bordas, margens e sombras.

O atributo width e height são mais fáceis de usar e entender, mas só podem definir a largura e a altura em pixels.

Imagens em outra pasta

Se você tiver suas imagens em uma subpasta, deverá incluir a pasta no caminho da imagem:

Exemplo

<img src="imagens/purple swirling star wallpaper.jpg" alt"purple swirling star wallpaper">

Imagens em outro servidor/site

Alguns sites apontam para para uma imagem em outro servidor.

Para fazer isso, você deve usar o URL absoluto da imagem:

Exemplo

<img src="https://pixabay.com/pt/photos/neuschwanstein-castelo-bav%C3%A1ria-701732/" alt"Castelo da Baviera"

Nota sobre imagens externas: As imagens exeternas podem estar em direitos autorais. Se você não obtiver permissão para usá-la, poderá estar violando leis de direitos autorais. Além disso, você não pode controlar imagens externas; elas podem de repente ser removidas ou alteradas

Imagens em movimento

As imagens em movimento podem ser criadas usando o elemento <img> com um arquivo de imagem em movimento (GIF) como fonte:

Exemplo

<img src="/VS-code/HTML/imagens.html/gifs/SFnN.gif" alt"personagem dos simpsons indo de um lado pro outro">

Imagem como link

Você pode usar uma imagem como um link, envolvendo a tag <img> com a tag <a>:

Exemplo

<a href="https://br.pinterest.com/pin/139682025938414357/"><img src="/VS-code/HTML/imagens.html/lírio aranha roxo, vermelho e azul.jpg" alt""></a>

Imagem flutuante

Você pode usar o atributo style para fazer uma imagem flutuar à esquerda ou à direita do texto:

Exemplo

<img src="/VS-code/HTML/imagens.html/purple swirling star wallpaper.jpg" alt="purple swirling star wallpaper" style="float: right; width: 100px;"

O atributo style neste exemplo faz a imagem flutuar à direita do texto.

Formatos de imagens comuns

  • JPEG - O formato de imagem mais comum. Suporta milhões de cores. Ideal para fotografias e imagens com muitos detalhes. Não suporta transparência.
  • PNG - Suporta milhões de cores. Ideal para imagens com áreas transparentes, como logotipos. Suporta transparência.
  • GIF - Suporta apenas 256 cores. Ideal para imagens simples, como gráficos e logotipos. Suporta transparência e animação.
  • SVG - Formato de imagem vetorial. Ideal para logotipos e ícones que precisam ser redimensionados sem perder qualidade. Suporta transparência.
  • WebP - Formato de imagem moderno que oferece melhor compressão e qualidade em comparação com JPEG e PNG. Suporta transparência e animação.

Nota: Nem todos os navegadores suportam todos os formatos de imagem. JPEG, PNG e GIF são amplamente suportados. SVG e WebP são suportados pelos navegadores modernos, mas podem não ser suportados por navegadores mais antigos

  • Você pode usar os atributos width e height ou o atributo style para definir o tamanho da imagem.
  • Você pode usar uma imagem como um link, envolvendo a tag <img> com a tag <a>.
  • Você pode fazer uma imagem flutuar à esquerda ou à direita do texto usando o atributo style.
  • Os formatos de imagem mais comuns são JPEG, PNG, GIF, SVG e WebP.
