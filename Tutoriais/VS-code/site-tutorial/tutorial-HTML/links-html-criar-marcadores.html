Links HTML - Criar Marcadores

Os links HTML podem ser usados para criar marcadores, para que os leitores possam pular para partes específicas da web.

Criar um marcador em HTML

Os marcadores podem ser úteis se uma página da web for muito longa.

Para criar um marcador - primeiro crie o marcador e, em seguida, adicione um link para ele.

Quando o link é clicado, a página rola para baixo ou para cima até o local do botão indicador.

Exemplo

Primeiro, use o atributo id para criar um indicador:

<h2 id="C4">Capítulo 4</h2>

Em seguida, adicione um link para o marcador ("Ir para o capítulo 4")

Exemplo

<a href="#C4">Pular para o capítulo 4

Você também pode adicionar um link para um marcador em outra página:

<a href="html-demo.html#C4">Pular para o capítulo 4</a>

Resumo do capítulo

  • Use o atributo id para criar um marcador.
  • Use o atributo href para criar um link para o marcador.
  • Você pode criar um link para um marcador em outra página.
