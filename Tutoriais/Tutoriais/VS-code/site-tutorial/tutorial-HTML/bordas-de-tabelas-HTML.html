Bordas de Tabelas HTML

As bordas são um dos elementos fundamentais na estilização de tabelas em HTML

Elas proporcionam estrutura, destaque visual e melhor compreensão de dados apresentados.

No final teremos uma tabela resumo com os principais atributos e seus efeitos visuais (para uma consulta rápida).

O Papel das Bordas em Tabelas HTML

As bordas das tabelas desenpenham funções além do mero visual. Elas separam visualmente os dados, facilitam a leitura. destacam agrupamentos e oferecem feedback visual em interfaces. Com CSS, é possível:

  • Definir bordas para toda a tabela, ou para células e cabeçalhos
  • Variar espessura, cor, estilo e raio de cada canto das bordas
  • Criar efeitos de zebra, realce ao passar o mouse, ou destacar células/ligações específicas
  • Adaptar a aparência para diferentes dispositivos e navegadores
  • Melhorar a acessibilidade e a semânticas do conteúdo

O domínio das propriedades a seguir é fundamental para criar experiêcias visuais tanto elegantes quanto funcionais.

Atributo HTML border - O Começo e Limitações

Como Usar

Historicamente, o atributo border em HTML foi a principal forma de adicionar bordas a uma tabela:

CSS

<table border=valor">"1">

    <tr>

        <th>Produto

        <th>Preço

    </tr>

    <tr>

        <td>Café

        <td>R$8,00

    </tr>

</table>

  • Define uma borda externa de largura igual aorvalor (em pixels).
  • Valor valor">0 remove completamente as bordas.
Limitações Importantes:
  • Não permite estilização individual (cor ou estilo).
  • NÃO é suportado em HTML5 (obsoleto; para código moderno, use CSS).
  • Não oferece controle sobre as bordas internas nem para células específicas

Substituição Moderna com CSS

Para um controle visual e responsividade, deve-se usar CSS

CSS

table {

    border: valor">1px solid #444;

}

Assim, você obtém o equivalente do atributo HTML, porém já abilitando mais personalização

Propriedade CSS border - Espessura, Estilo e Cor

Estrutura da Propriedade border

A propriedade border permite controlar:

  • Largura: border-width (ex: 1px, 2em, thin, thick)
  • Estilo: border-style Ex:
    • solid
    • dashed
    • dotted
    • double
    • groove
    • ridge
    • inset
    • outset
    • none
    • hidden
  • Cor: border-color (ex: #0055ff, red, rgba(255, 0, 0, 0.3)

Podem ser usadas separadamete (border-top, border-left, etc.) ou em conjunto:

CSS

border: valor">2px dotted #9b59b6;

/ * ou */

border: valor">2px;

border: valor">dotted;

border: valor">#9b59b6;

Exemplo Prático (Tabela Básica)

HTML

<style>

table, th, td {

    border: valor">1px solid #333;

}

/style>

<table>

    <tr>

        <th>Aluno<th>

        <th>Nota<th>

    </tr>

    <tr>

        <td>Joana<td>

        <td>10<td>

    </tr>

</table>

Com o CSS acima, cada célula e o contorno da tabela terão uma borda sólida de 1px, preta. Note que isso produz, por padrão, linhas duplas entre as células(elementos th/ th e a borda da tabela)

Variação de Lados

É possível criar bordas diferentes em cada lado:

CSS

td {

    border-top:valor"> 1px solid #00f;

    border-bottom:valor"> 2px dashed #f00;

    border-left:valor"> none;

    border-right:valor"> 4px double #0f0;

}

Este recurso é útil para criar efeitos visuais sofisticados e destacar partes específicas da tabela.


Removendo as Bordas Duplas:

border-collapse

Por padrão, ao definir bordas em table, th, td, as linhas se sobrepõem e aparecem duplas. Para corrigi-lo, utiliza-se border-collapse

Modos da Propriedade

  • border-collapse: valor">separate(padrão): Células com bordas individuais (aparecem bordas duplas).
  • border-collapse: valor">collapse: Bordas sobrepostas viram uma só (visual limpo e moderno).

CSS

table, th, td {

    border: valor">1px solid #222;

    border-collapse: valor">collapse;

}

O resultado é uma tabela com linhas simples entre células e visual unificado.

HTML

<table style="border-collapse: valor">separate;">

<!-- Bordas duplas (padrão HTML) -->

...

</table>

<table style="border-collapse: valor">collapse;">

<!-- Bordas únicas compartilhadas pelas células -->

...

</table>

Resumo: Prefira sempre border-collapse: valor">collapse para tabelas de dados — este é o padrão visual moderno e elimina facilmente as linhas duplas, além de ser suportado por todos os navegadores modernos.

Espaçamento entre Células: border-spacing

Quando se utiliza border-collapse: valor"> separate, é possível controlar o espaço entre as células com border-spacing:

CSS

table {

    border-collapsevalor"> separate;

    border-spacingvalor"> 10px 20px; /* Horizontal 10px, Vertical 20px */

}

  • O primeirorvalor é o espaçamento horizontal; o segundo, vertical.
  • Essencial para designs que requerem separação visual entre células.
  • Importante: border-spacing não faz efeito se a propriedade border-collapse estiver setada como valor">collapse

CSS

table {

    border-collapse: valor">separate;

    border-spacing: valor">8px 16px;

    border: valor">1px solid #444;

}

td {

    border: valor">1px solid #444;

    text-alignvalor">

}

Esta configuração aplica espaços entre células, além da borda ao redor de cada célula.

Controle fino: border-style, border-width e border-style

Define o visual da linha: sólida, pontilhada, tracejada, dupla, 3D, oculta, etc.

CSS

th, td {

border-style: valor">dashed; /*Possíveis valores:

  • solid

  • dashed

  • dotted

  • double

  • groove

  • ridge

  • outset

  • inset

  • none

  • hidden

}

border-width

Controla a espessura da borda. Aceita de 1 a 4 medidas (em px, em, %, etc)

td {

border-width: valor">2px 4px; /* 2px para topo/baixo, 4px para esquerda/direita */

border-color

Define as cores da borda, podendo ser distintas para cada lado:

CSS

td {
    border-color:valor">
    border-color: valor">red green blue yellow; /* topo, direita, baixo, esquerda */
}

Estas três propriedades podem ser usadas juntas, permitindo customização avançada. Também aceitam valores individuais em border-top-width, border-right-style etc. A personalização é plena.

Bordas Arredondadas: border-radius

A propriedade border-radius aplica cantos arredondadas às bordas de elementos, inclusive tabelas. Sua sintaxe permite flexibilidade para criar desde círculos até raios distintos em cada canto:

CSS

table {

    border: valor">2px solid #777;

    border-radius: valor">12px;

}

Valores Individuais

table {

    border-radius: valor"> 10px 25px 30px 5px; /* topo-esquerda, topo-direita, inferior-direita, inferior-esquerda */

}

Observações Importantes

  • Ap usar border-collapse, o arrendondamento pode não funcionar corretamente em todos os navegadores para bordas internas/externas.
  • Para garantir bordas arredondadas sem bug visual, use frequentemente border-collapse: separate junto com overflow: valor">hidden.

HTML

<style>

table {

    border: valor">1px solid #888;

    border-radius: valor">15px;

    border-collapse: valor">separate;

    overflow: valor">hidden;

}

td, th {

    border: valor">1px solid #888;

}

</style>

<table>

    <tr>

        <th>Nome </th>

        <th>Cidade </th>

    </tr>

    <tr>

        <td>Ana </td>

        <td>Rio </td>

    </tr>

</table>

Nas bordas arredondadas, o design se torna mais amigável e moderno

Bordas com imagens: border-image

A propriedade border-image traz possibilidades criativas, subtituindo a linha sólida da borda por uma imagem. Ideal para molduras ricas ou visuais lúdicos:

Sintaxe Simples

CSS

table{

    border:

    border-image: url(/VS-code/CSS/imagens.css/bordas/vecteezy_border_1195178.png);

}

  • border-image-source: O arquivo de imagem (ou gradiente).
  • border-image-slice: Onde a imagem é cortada para formar cantos e partes laterais.
  • border-image-width: Largura da borda.
  • border-image-repeat: como repetir a imagem (stretch, repeat, round, space)

Dica: Defina sempre border-style, pois alguns navegadores não exibem a imagem se faltar o estilo ou largura.

Observações Especiais

  • border-image não funciona em células internas (td/th) quando border-collapse está como collapse- use em tabelas com border-collapse: separate ou apenas na borda externa da tabela
  • Não suporta cantos arredondados(border-radius): para efeitos visuais com cantos arredondados, crie bordas do do tipo imagem já com os cantos arredondados

Exemplo Prático

CSS

table {

    border: 12px solid #fff;

    border-image: url(border-frame.png) 30 round;

    border-collapse: separate;

}

Este recurso é pouco usado em tabelas de dados, mas pode destacar quadros especiais ou galerias de imagens.

Diferença entre border-collapse e border-spacing

A distinção entre essas propriedades é fundamental para uma estilização de tabelas sem surpresas:

  • border-collapse: collapse; elimina qualquer espaçamento entre células e colapsa as bordas duplas em uma só. Não é possível utilizar border-spacing nesse modo
  • border-collapse: separate; exibe as bordas de cada célula independentemente e permite o uso de border-spacing para adicionar espaço entre elas

Exemplo visual

Exemplo visual

/* com collapse - bordas unidas */

table{

    border-collapse: collapse;

}

/* com separate e espaçamento */

table{

    border-collapse: collapse;

    border-spacing: 12px 6px; /* horizontal | vertical */

}

Dica Importante: O efeito visual é completamente diferente e impacta tanto na clareza quanto no espaço utilizado pela tabela em tarefas pequenas.
