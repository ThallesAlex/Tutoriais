Colspan e Rowspan

O que é colspan?

  • colspan faz uma célula ocupar várias colunas
  • Usado você quer que uma célula se estenda horizontalmente
Exemplo:

HTML

<table border= "1">

    <tr>

        <th colspan="2">Nome Completo </th>

        <th>Idade </th>

    <tr>

<tr>

        <td>João</td>

        <td>Silva</td>

        <td>25</td>

    <tr>

</table>

Aqui, a célula "Nome completo" ocupa duas colunas: "João" e "Silva".

O que é rowspan?

  • rowspan faz uma célula ocupar várias linhas
  • Usado quando você quer uma célula se estenda verticalmente

HTML

<table border= "1">

    <tr>

        <th rowspan="2">Telefone</th>

        <th>555-1234 </th>

    <tr>

    <tr>

        <td>555=5678</td>

    <tr>

</table>

Aqui, a célula "Telefone" ocupa duas linhas, enquanto os números aparecem ao lado em linhas separadas.

Dicas Importantes

  • Use colspan e rowspan com moderação para evitar tabelas confusas.
  • Verifique se o número total de colunas e linhas permanece consistente após usar esses atributos.
  • Combine com CSS para melhorar a aparência e legibilidade da tabela.
