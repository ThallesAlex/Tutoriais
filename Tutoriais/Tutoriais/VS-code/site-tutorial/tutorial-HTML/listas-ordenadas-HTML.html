Listas Ordenadas (<ol>)

Listas ordenadas são usadas quando a sequencia dos itens é fundamental. Podem representar etapas de um processo, hierarquias, rankings, classificações, agendas, entre outros.

Quando renderizadas, cada item é precedido por um número, letra ou outro índice, automaticamente gerado pelo navegador.

Estrutura Básica

A estrutura mínima de uma lista ordenada é:

HTML

<ol>
   <li>Primeiro passo</li>
   <li>Segundo passo</li>
   <li>Terceiro passo</li>
</ol>

O navegador exibirá por padrão 1, 2, 3, 4, etc. antes de cada <li>

HTML

<ol>
   <li> Abrir o programa </li>
   <li> Carregar o documento</li>
   <li> Editar o texto</li>
   <li> Salver as alterações</li>
</ol>

Este exemplo deixa claro que a ordem importa, já que cada passo depende da anterior.

Personalização de marcadores com atributos HTML

O elemento <ol> possui alguns atributos de personalização:

  • type: define o tipo da numeração: números (1, padrão), letras maiúsculas(A), letras minúsculas(a), algarismos romanos maiúsculos(I) ou minúsculos(i).
  • start: indica a partir de qual valor a numeração ou senquência começa
  • reversed: Inverte a ordem, numerando do maior para o menor.

Exemplos

Com letras maiúsculas

HTML

<ol type="A">
   <li>Planejamento</li>
   <li>Desenvolvimento</li>
   <li>Testes</li>
</ol>

Saída:
  1. Planejamento
  2. Desenvolvimento
  3. Testes

Começando do 5

HTML

<ol start="5">
   <li>Quinto item</li>
   <li>Sexto item</li>
</ol>

Saída
  1. Quinto item
  2. Sexto item

lista ordenada decrescente

HTML


<ol reversed>
   <li>Terceiro lugar</li>
   <li>Segundo lugar<li>
   <li>Primeiro</li>
</ol>

Saída
  1. Terceiro lugar
  2. Segundo lugar
  3. Primeiro lugar
Você pode combinar type , star e reversed para criar listas ainda mais flexíveis

Listas Ordenadas aninhadas e mistas

Também é possivel criar listas aninhadas, seja de mesmo tipo ou misturando listas ordenadas e não ordenadas:

HTML

<ol>
   <li>Introdução</li>
   <li>Procedimentos
      <ul>
         <li>Verificação inicial</li>
         <li>Execução</li>
      </ul>
   </li>
   <li>Conclusão</li>
</ol>

Saída
  1. Introdução
  2. Procedimentos
    • Verificação inicial
    • Execução
  3. Conclusão
Neste exemplo, o segundo item da lista ordenada contém uma sublista não ordenada

Itens de lista numerados individualmente

Além dos atributos de <ol>, o <li> aceita o atributo value (pricipalmente útil em listas ordenadas), que permite reiniciar o marcador em um número específico para aquele item:

HTML

<ol>
   <li>Primeiro</li>
   <li value="10">Dez</li>
   <li>Onze</li>
</ol>

Saída
  1. Primeiro
  2. Dez
  3. Onze
O próximo <li> continua a partir do valor definido anteriormente
