HTML Parágrafos

Um parágrafo sempre começa em uma nova linha e geralmente é um bloco de texto.

Parágrafos HTML

O elemnto HTML <p> define um parágrafo.

Um parágrafo sempre começa em uma nova linha, e os navegadores adicionam automaticamente algum espaço em branco (uma margem) antes e depois de um parágrafo.

Um parágrafo pode conter textos, imagens, links e outros elementos HTML.

Exemplo

<p>Isso é um parágrafo.<p>

<p>Isso é outro parágrafo.<p>

Exibição HTML

você não pode ter certeza de como o HTML será exibido.

Telas grandes ou pequenas criarão resultados diferentes.

Com o HTML, você não poe alterar a exibição adicionando espaços ou linhas extras no seu código HTML

O navegador removerá automaticamente quaisquer espaços e linhas extras quando a página for exibida:

Exemplo

<p>Isso é um parágrafo. Isso é ainda parte do mesmo parágrafo. Há muitos espaços na linha de código deese parágrafo mas o navegador ignora isso </p>

<p>Isso é outro parágrafo.</p>

Regras horizontais HTML

A tag <hr> define uma quebra temática em uma página HTNL e é mais frequentemente exibida como uma regra horizontal.

O elemento <hr> é usado para separar conteúdo (ou definir uma alteração) em uma página HTML

Tipo essa linha aqui debaixo

O elemento <hr> é um elemento vazio, o que significa que ele não tem uma tag de fechamento.

Quebras de linha

O elemnto HTML <br> define uma quebra de linha.

Use o <br> se você quiser uma quebra de linha (uma nova linha) sem iniciar um novo parágrafo

Exemplo

<p>Isso é um parágrafo.<br>Isso é uma nova linha na mesma parágrafo.</p>

<p>Isso é outro parágrafo.</p>

saída

Isso é um parágrafo.
Isso é uma nova linha na mesma parágrafo.

Isso é outro parágrafo.

O elemento <br> é um elemento vazio, o que significa que ele não tem uma tag de fechamento.

O problema do poema

Este poema será exibido em uma única linha

Exemplo

<p>
Minha Bonnie está no oceano.
Minha Bonnie está no mar.
Minha Bonnie está em algum lugar sobre o oceano.
Oh, traga minha Bonnie de volta para mim
</p>

saída

Minha Bonnie está no oceano. Minha Bonnie está no mar. Minha Bonnie está em algum lugar sobre o oceano. Oh, traga minha Bonnie de volta para mim

Solução

O elemento HTML <pre> define um texto pré-formatado

O texto dentro de um elemento <pre> é exibido em uma fonte de largura fixa(geralmente Courier) e preserva espaços e quebras de linha:

Exemplo

<pre>
Minha Bonnie está no oceano.
Minha Bonnie está no mar.
Minha Bonnie está em algum lugar sobre o oceano.
Oh, traga minha Bonnie de volta para mim
</pre>

saída

            Minha Bonnie está no oceano.
            Minha Bonnie está no mar.
            Minha Bonnie está em algum lugar.
            Oh, traga minha Bonnie de volta para mim
