Listas HTML

Introdução

Compreender as listas em HTML é um dos passos mais essenciasis e iniciais ao aprender desenvolvimento Web.

As listas tornam a apresentação de informações mais clara, organizada e acessível. Seja para destacar etapas de um processo, apresentar um menu de navegação, exibir especificações técnicas ou criar glossários.

O HTML oferece três tipos principais de listas: não ordenadas(<ul>), ordenedas(<ol>) e de definição (<dl).

Cada uma dessas listas possui uma função semântica especifica, baseia-se em uma estrutura de marcação própria, pode ser personalizada visualmennte com CSS e, quando criada com boas práticas, contribui para a acessibilidade e Seo do site
