Título da página - HTML

O que é o título da página?

O título da página é o texto que que aparece:

  • Na aba do navegador quando você abre um site
  • Na barra de título do navegador(em alguns casos)
  • Como o nome da página nos resultados de busca do Google e outros buscadores
  • Quando você adiciona a página aos favoritos ou bookmarks

Como usar a tag <title>?

Ela é colocada dentro da seção <head> do seu documento HTML. Exemplo:

Exemplo

<head>

<title>Minha Página Incrível</title>

</head>

Regras importantes

  • O título deve ser descritivo e relevante ao conteúdo da página
  • Evite títulos muito longos; o ideal é entre 50-60 caracteres
  • Use palavras-chave importantes para melhorar o SEO (otimização para motores de busca)
  • Cada página do seu site deve ter um título único
