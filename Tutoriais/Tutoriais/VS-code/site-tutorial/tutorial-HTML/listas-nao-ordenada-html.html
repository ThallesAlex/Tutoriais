Listas Não Ordenada (<ul>)

Listas não ordenadas agrupam itens onde a ordem não é relevante para a compreensão do conteúdo. Tais listas são renderizadas usando marcadores gráficos (bulles), geralmente círculos, quadrados ou círculos, vazados, indicando apenas que os itens fazem parte de um mesmo grupo, mas sem priorização ou sequencia clara entre eles

Estrutura Básica

A estrutura mínima de uma lista não ordenada é

HTML

<ul>

   <li>Primeiro Item </li>

   <li>Segundo Item </li>

   <li>Terceiro item </li>

</ul>

Cada item da lista é envolvido pela tag <li> (List item). O navegador, por padrão, exibirá um marcador (geralmente um círculo preto) antes de cada item

HTML

  • Pão
  • Leite
  • Queijo
  • Tomate

Este exemplo mostra o uso típico para uma lista não ordenada: itens de uma lista de compras, onde a ordem não tem importância.

Lista não ordenada com links

Listas <ul> são amplamente usadas para menus de navegação, já que a ordem dos links não precisa ser sequêncial:

HTML

<nav>
   <ul>
      <li><a href="#home">Início</a></li>
      <li><a href="#produtos">Produtos</a></li>
      <li><a href="#contato">Contato</a></li>
   </ul>
</nav>

Aqui, cada item é um link, facilitando a navegação do usuário.

Personalização de marcadores com HTML

Embora o ideal seja personalizar os marcadores via CSS, o HTML ainda suporta um atributo de tipo obsoleto (type), que permite escolher entre disco, círculo ou quadrado em alguns navegadores antigos:

HTML


<ul type="square">
   <li> Pão</li>
   <li> Leite</li>
   <li> Queijo</li>
</ul>
Porém, para projetos modernos, utilize sempre CSS para controlar marcadores, pois o atributo type foi depreciado e não é mais recomendade

Listas não ordenadas aninhadas (listas dentro de listas)

Você pode criar sub-listas, agrupando itens relacionados:

HTML


<ul>
   <li>Frutas
      <ul>
         <li>Maçâ</li>
         <li>Banana</li>
      </ul>
   </li>
   <li>Legumes
      <ul>
         <li>Cenoura</li>
         <li>Batata</li>
      </ul>
   </li>
</ul>

O navegador irá exibir um recuo visual para os subitens e pode variar o tipo de marcador nos níveis internos
