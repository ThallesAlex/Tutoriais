Em HTML, o atributo class é usado para agrupar elementos com um mesmo nome, permitindo aplicar estilos CSS ou manipular elementos com JavaScript de forma eficiente.

O que é o atributo class?

O atributo class é um atributo global em HTML, ou seja, pode ser usado em praticamente qualquer elemento. Ele serve para atributir um ou mais nomes de classe a um elemento, separados por espaçoes.

HTML

<div class="caixa destaque">Conteúdo aqui</div>

Nesse exemplo, o elemento <div> pertence às classescaixa e destaque

Para que serve uma classe?

  1. Estilização com CSS

    Você pode aplicar estilos específicos a elementos com determinadas classe:

    CSS


    .destaque
    background-color: yellow;
    font-weight: bold;

  2. Manipulação com JavaScript

    E possível selecionar elementos por classe e manipulá-los:

    JavaScript


    const elementos = document.getElementByClassName(destaque);

  3. Organização e Reutilização

    Usar classe permite reutilizar estilos e comportamento em vários elementos, mantendo o código mais limpo e modular.

Boas práticas

  1. Use nomes descritivos e semânticos, como botão-enviar em vez de vermelho
  2. Evite nomes genéricos como box1 ou estilo2
  3. Reutilize classes sempre que possível para manter consistência visual e funcional.

Diferença entreclass e id

Atributo Pode ser udado em vários elementos? Usado para...
class Sim Estilizar e agrupar elemetos
id Não (único por página) Identificar um elemento específico
