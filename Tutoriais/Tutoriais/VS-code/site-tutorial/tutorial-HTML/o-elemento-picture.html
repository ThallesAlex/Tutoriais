O elemento Picture

O que é o <picture>

O <picture> é um contêiner que permite definir múltiplas fontes de imagem para um único <img>. O navegador escolhe automaticamente a imagem mais adequada com base em:

  • Tamanho da tela
  • Resolução do dispositivo
  • Suporte ao formato da imagem (como WebP, JPEG, PNG)

Sintaxe básica

Exemplo básico:

<picture>
    <source srcset="imagem-alta-resolucao.webp" media="(min-width: 800px)" type="image/webp">
    <source srcset"imagem-baixa-resolucao.jpg" media="(max-width: 799px)" type="image/jpeg">
    <img src="imagem-padrao.png" alt"Descrição da imagem">
</picture>

Explicação

  • source: define uma imagem alternativa com base em uma condição (media)
  • media: usa media queries pra definir quando aquela imagem deve ser usada
  • srcset: caminho da imagem.
  • <img>: imagem padrão, usada se nenhuma <source> for compatível.

Quando usar

  • Para fornecer imagens responsivas que se adaptam a diferentes tamanhos de tela.
  • Para otimizar o desempenho, carregando imagens menores em dispositivos com telas menores.
  • Para usar formatos de imagem modernos (como WebP) quando suportados pelo navegador.

Exemplos com formatos diferentes

Exemplo com WebP e JPEG:

<picture>
    <source srcset"imagem.webp" type"image/webp">
    <source srcset"imagem.jpg" type"image/jpeg">
    <img src"imagem.png" alt"Descrição da imagem">
</picture>
