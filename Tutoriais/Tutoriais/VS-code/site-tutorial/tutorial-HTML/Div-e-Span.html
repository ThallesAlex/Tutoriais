Div e Span

Os elementos <div> e <span> são usadas para agrupar e organizar conteúdo HTML, mas têm propósitos diferentes: <div> é para blocos e <span> para trechos em linha.

<div> - Elemento de Bloco

  • Função principal: Agrupar elemenetos em nível de bloco.
  • Comportamento: Ocupa toda a largura disponível, inicíado em uma nova linha.
  • Uso comum: Estruturar seções da pagina como cabeçalhos, rodapés, colunas, caixas de conteúdo.
  • Pode conter: Parágrafos, listas, imagens, outros <div>, etc.

Exemplo

HTML

<div class="seção-pricipal">
<h1>Bem vindo</h1>
<p>Este é o conteúdo da seção</p>
</div>

<span> - Elemento em linha

  • Função principal: Agrupar conteúdo fraseado dentro de uma linha.
  • Conportamento: Não quebra linha, se comporta como parte do texto.
  • Uso comum: Aplicar estilos ou identificar partes específicas de um texto.
  • Pode conter: Texto e elementos em linha como <a><strong>, etc.

Exemplo

HTML

<p>Este é um <span class="destaque"texto destacado</span> dentro do parágrafo.

Diferençãs principais

Características div span
Tipo de elemento Bloco Em linha
Quebra de linha Sim Não
Uso típico Estrutura de layout Estilização de texto
Pode conter Elementos de bloco e em linha Apenas elementos em linha

Aplicações com CSS

Ambos são frequentemente usados com classes ou IDs para aplicar estilos:

HTML

<div id="conteiner" class="box">
<span class="negrito">Texto</span>

CSS

.box
backgroud-colo: #f0f0f0;
padding: 20px;

.negrito
font-weight: bold;
