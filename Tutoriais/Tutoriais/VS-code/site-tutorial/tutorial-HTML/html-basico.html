Exemplos básicos de HTML

Nesse capítulo, mostraremos alguns exemplos básicos de HTML.

Não se preocupe se usarmos tags que você ainda não aprendeu.

Documentos HTML

Todos os documentos HTML devem começar com uma declaração de tipo de documento:<!DOCTYPE html>

O próprio documento HTML começa com: <html> </html>

A parte visível de documento HTML está entre: <body>/body>

A declaração <DOCTYPE>

A declaração <DOCTYPE> representa o tipo de documento e ajuda os navegadores a exibir as páginas da Web corretamente

A declaração <DOCTYPE> não diferencia maiúsculas de minúsculas

A declaração <DOCTYPE> diz para o navegador a versão HTML, no caso o mais recente é o HTML5 (data: 06/10/2025)

Cabeçalhos HTML

Os cabeçalhos HTML são definidos com as tags de <h1> a <h6>

O h1 é o mais importante, o h2 o segundo, h3 o terceiro, e por aí vai.

Imagens HTML

As imagens HTML são definidas com a tag <img>.

O arquivo de origem(), e o texto alternativo(),são fornecidos com os atributos src e alt.

Exemplo

<img src="HTML/imagens.html/O penitente.jpeg" alt="o penitente" width"104" height ="142"
