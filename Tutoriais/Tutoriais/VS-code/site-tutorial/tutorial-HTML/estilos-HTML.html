HTML Estilos

O atributo HTML style é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

Exemplo

Este parágrafo é azul.

Este parágrafo é vermelho.

Este parágrafo é verde.

O atributo de estilo HTML

Definir o estilo de um elemento HTML pode ser feito com o atributo style

O atributo HTML style tem a segunte sintaxe:

<nome da tag style="propriedade:valor;>"

A propriedade é uma propriedade CSS. O valor é um valor CSS

Cor de fundo

A propriedade CSS background-color define a cor de fundo para um elemnto HTML.

Exemplo

defindo a cor de fundo para preto e a letra para branco

<p style="background-color:black; color:white;"> Este parágrafo tem a cor branca e o fundo preto

<p style="background-color:black; color:white;"> Este parágrafo tem a cor branca e o fundo preto

<p style="background-color:black; color:white;"> Este parágrafo tem a cor branca e o fundo preto

Exercício

Use o atributo de estilo HTML para definir a cor do fundo para azul, vermelho e verde, respectivamente.

Exemplo

<p style="background-color:blue;">Este parágrafo tem o fundo azul.

<p style="background-color:red;">Este parágrafo tem o fundo vermelho.

<p style="background-color:green;">Este parágrafo tem o fundo verde.

Cor de texto

Exemplo

A propriedade CSS color definne a cor do texto para um elemnto HTML:

<p style="color:blue;">Este parágrafo é azul.

<p style="color:red;">Este parágrafo é vermelho.

<p style="color:green;">Este parágrafo é verde.

Fontes

A propriedade CSS font-family define a fonte usada a um elemento HTML:

Exemplo

<p style="font-family:Arial;">Este parágrafo usa a fonte Arial.

<p style="font-family:Courier;">Este parágrafo usa a fonte Courier.

<p style="font-family:Georgia;">Este parágrafo usa a fonte Georgia.

<p style"font-family:Times New Roman;">Este parágrafo usa a fonte Times New Roman.

<p style"font-family:Verdana;">Este parágrafo usa a fonte Verdana.

Tamanho do texto

A propriedade CSS font-size define o tamanho da fonte usada em um elemento HTML:

Exemplo

<p style="font-size:16px;">Este parágrafo tem um tamanho de fonte de 16 pixels.

<p style="font-size:20px;">Este parágrafo tem um tamanho de fonte de 20 pixels.

<p style"font-size:24px;">Este parágrafo tem um tamanho de fonte de 24 pixels.

<p style"font-size:28px;">Este parágrafo tem um tamanho de fonte de 28 pixels.

<p style"font-size:32px;">Este parágrafo tem um tamanho de fonte de 32 pixels.

Alinhamento do texto

A propriedade CSS text-align define o alinhamento do texto em um elemento HTML:

Exemplo

<p style="text-align:left;">Este parágrafo está alinhado à esquerda.

<p style="text-align:center;">Este parágrafo está centralizado.

<p style"text-align:right;">Este parágrafo está alinhado à direita.

<p style"text-align:justify;">Este parágrafo está justificado
