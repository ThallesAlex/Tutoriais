Elementos HTML

Um elemento HTML é definido por uma marca inicial, algum conteúdo e uma tag no final.

Elementos HTML

O elemento/tag HTML é tudo, desde a tag inicial até a tag final.

Elementos HTML aninhados

Elementos HTML podem ser aninhados (isso significa que os elementos podem conter outros elementos).

Todos os documentos HTML consistem em elementos HTML aninhados.

O exemplo a seguir contém 4 elementos HTML aninhados: <html> <body> <h1> <p><html>

Exemplo

<!DOCTYPE>

<html>

<body>


<h>isso é um título

<p>isso é um parágrafo


</body>

</html>

Exemplo explicado

O elemento <html>, é o elemento raiz e difine todo o documento HTML.

Tem uma tag inicial <html> e uma tag final </html>.

Então, dentro do elemento <html> há um alemento <body>.



<body>


<h1>Isto é um título</h1>

<p>Isto é um parágrafo</p>


</body>

O elemento <body> define o corpo do documento

Tem uma tag inicial, e uma tag final

Então dentro do elemento <body> há outros 2 elementos, h1 e p.

O elemento <h1> define um título, ele tem uma tag inicial e uma final

O elemento <p> define um parágrafo, ele tem uma tag inicial e uma final


Nunca pule a tag final

Alguns elementos HTML serão exibidos corretamente, mesmo se você esquecer a tag final.

No entanto nunca confie nisso! Resultados inesperados e erros podem acontecer se você esquecer a end tag!

Elementos HTML vazios

Elementos HTML sem conteúdo são chamados de elementos vazios.

A tag <br>define uma quebra de linha, e é um elemento vazio. Não tem tag de fechamento

Exemplo

<p>Isto é um parágrafo com
<br> quebra linha

HTML não diferencia maiúsculas de minúsculas

As tags HtML náo diferenciam maiúsculas de minúsculas: significa que <P> é o mesmo que <p>
