Elementos Block(bloco)

Esses elementos ocupam toda a largura disponível da página, criando uma "quebra de linha" antes e depois deles. Eles são usados para estruturar sexções maiores de conteúdo

Exemplos comuns:

  • span.tag{<div>}: usado para agrupae outros elementor
  • span.tag{<p>}: parágrafo
  • <h1> a <h6>: títulos
  • <ul>, <ol>, <li>: listas

Elementos Inline(em linha)

Esses elementos ocupam o espaço necessário e não criam quebra. Eles são usados dentro de blocos para formatar ou adicionar conteúdo específico

Exemplos comuns:

  • <span>: usado para aplicar estilos ou agrupar texto
  • <a>: links
  • <strong>, <em>: destaque de texto
  • <img>: imagens

Diferença Visual

Imagine que você tem um parágrafo com uma palavra em negrito. O parágrafo (<p>) é block, e a palavra em negrito (<strong>) é inline - ela aparece dentro da linha do texto, sem quebrar.
