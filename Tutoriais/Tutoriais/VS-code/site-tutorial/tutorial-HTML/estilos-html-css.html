Estilos HTML - CSS

CSS significa Cascading Style Sheets

CSS economiza muito trabalho. pode controlar o layout de vária páginas da web de uma só vez.

O que é CSS?

As folhas de estilo em cascata (CSS) são usadas para formatar o layout de uma página da Web.

Com CSS, você pode controlar a cor, a fonte, o tamanho do texto, o espaçamento entre os elementos, como os elementos são posicionados e dispostos, qual o plano de fundo imagens ou cores de fundo devem ser usadas, exibições diferentes para dispositivos diferentes e tamanhos de tela e muito mais!

Ponto: A palavra cascata significa que um estilo aplicado a um elemento pai também se aplicará a todos os elementos filhos dentro do pai. Portanto, se você definir a cor do corpo do texto como "azul", todos os títulos, parágrafos e outros elementos de texto dentro do corpo também terão a mesma cor (a menos que você especifique outra coisa)!

Usando CSS

O CSS pode ser adicionado a documentos HTML de 3 maneiras:

  • Inline - usando o atributo styledentro de elementos HTML
  • Interno - usando um elemento <head>
  • Externo - usando um elemento <link> para vincular a um aequivo CSS externo

A maneira mais comum de adicionar CSS é manter os estilos em CSS externo. No entanto, neste tutorial, usaremos estilos embutidos e internos, porque isso é mais fácil de usar. demonstrar, e mais fácil para você experimentá-lo você mesmo.

CSS Inline

Um CSS inline é usado para aplicar um estilo exclusivo a um único elemento HTML.

Um CSS inline usa o atributo <style> de um elemento HTML

O exemplo a seguir define a cor do texto do elemento <h1> como azul, e a cor do texto do elemento <p> para vermelho:

Exemplo

<h1 style="color: blue;"> Um título azul </h1>

<p style="color: red;">um parágrafo vermelho</p>

CSS interno

Um CSS interno é usado para definir um estilo para uma página HTML

Um CSS é definido na seção <style> de uma página HTML, dentro de um elemento <head>

O exemplo a seguir define a cor do texto de TODOS os elementos <h1>(nessa página) para azul, e a cor do texto de TODOS os elementos <p> para vermelho. Além disso, a página será exibida com uma cor de fundo "powderblue":

<!DOCTYPE html>

<head>

<style>

body {background-color: powderblue;}

h1 {color: blue;}

p {color: red;}

</style>

</head>

<body>

<h1>Um título azul</h1>

<p>um parágrafo vermelho</p>

</body>

</html>

CSS externo

Uma folha de estilo externa é usada para definir o estilo de muitas páginas HTML.

Para usar uma folha de estilo externa,adicione um link para ela na seção <head> de cada página HTML.

O link é definido usando um elemento <link>, com os atributos rel e href:

<!DOCTYPE html>

<html>

<head>

<link rel="stylesheet" href="meu-estilo.css">

</head>

<body>


<h1>Um título azul</h1>

<p>um parágrafo vermelho</p>


</body>

</html>

A folha de estilo externa pode ser escrita em qualquer editor de texto. O arquivo não pode conter nenhum HTML e deve ser salvo com uma extensão .css

Aqui está a aparência do arquivo "styles.css":

"meu-estilo.css"

body {
background-color: powderblue
;}

h1 {
color: blue
;}

p {
color: red
;}

Ponto:Com uma folha de estilo externa, você pode alterar a aparência de um site inteiro, alterando um arquivo!

Cores, Fontes e Tamanhos CSS

Aqui, te mostrarei algumas propriedades CSS comumente usadas.

A propriedade CSS valor define a cor do texto a ser usada

A propriedade CSS font-family define a fonte a ser usada.

A propriedade CSS font-size define o tamanho do texto a ser usado.

Exemplo

Uso das propriedades CSS color, font-size e font-family

<!DOCTYPE html>

<html>

<head>

<style>

h1 {
color: blue;
font-family: verdana;
font-size: 300%;
}

</style>

</head>

<body>

<h1>Um título azul em Verdana, tamanho 300%</h1>

</body>

</html>

Borda CSS

A propriedade CSS border define uma borda em torno de um elemento HTML.

Ponto: Você pode definir uma borda para quase todos os elementos HTML.

p {
border: 2px solid powderblue;
}

Preenchimento CSS

A propriedade CSS padding define um Preenchimento(espaço) entre o texto e a borda.

p {
border: 2px solid powderblue;
padding: 20px;
}

Margem CSS

A propriedade CSS margin define uma margem (espaço) fora da fronteira.

p {
border: 2px solid powderblue;
margin: 50px

Link para CSS externo

As folhas de estilo externas podem ser referenciadas com um URL completo ou com um caminho relativo à página da Web atual.

Este exemplo usa uma URL completa para vincular a uma folha de estilo:

<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">

Este exemplo usa um caminho relativo para vincular a uma folha de estilo:

<link rel="stylesheet" href="/css/meu-estilo.css">

este exemplo é vinculado a uma folha de estilos localizada na mesma pasta que a página HTML:

<link rel="stylesheet" href="meu-estilo.css">
