HTML Formatação de texto

O HTML contém vários elementos para definir texto com um significado especial.

Exemplo

<b>Texto em negrito</b>

<strong>Texto importante</strong>

<i>Texto em itálico</i>

<em>Texto enfatizado</em>

<mark>Texto marcado</mark>

<small>Texto pequeno</small>

<del>Texto deletado</del>

<ins>Texto inserido</ins>

<sub>Texto subscrito</sub>

<sup>Texto sobrescrito</sup>

Elementos HTML <b> e <strong>

O elemento HTML <b> é usado para tornar o texto em negrito, sem indicar que é mais importante.

O elemento HTML <strong> é usado para definir texto com forte importância, e o navegador geralmente o exibe em negrito.

Exemplo

<b>Este texto é negrito</b>

<strong>Este texto é importante</strong>

Elementos HTML <i> e <em>

O elemento HTML <i> é usado para tornar o texto em itálico, sem indicar que é mais importante.

Dica: O <i> é frequentemente usado para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de um navio, etc.

O elemento HTML <em> é usado para definir texto enfatizado, e o navegador geralmente o exibe em itálico.

Dica: O <em> é frequentemente usado para indicar ênfase quando o texto é lido em voz alta,com um tom de estresse.

HTML Elementos <mark>, <small>, <del> e <ins>

O elemento HTML <mark> é usado para definir texto marcado ou destacado.

Exemplo

<mark>Este texto está marcado</mark>

O elemento HTML <small> é usado para definir texto pequeno.

Exemplo

<small>Este texto é pequeno</small>

O elemento HTML <del> é usado para definir texto que foi deletado de um documento.

Exemplo

<del>Este texto foi deletado</del>

O elemento HTML <ins> é usado para definir texto que foi inserido em um documento.

Exemplo

<ins>Este texto foi inserido</ins>

Os elementos HTML <sub> e <sup> são usados para definir texto subscrito e sobrescrito, respectivamente.

Exemplo

parágrafo <sub>subscrito</sub>

parágrafo <sup>sobrescrito</sup>

saída

parágrafo subscrito

parágrafo sobrescrito
