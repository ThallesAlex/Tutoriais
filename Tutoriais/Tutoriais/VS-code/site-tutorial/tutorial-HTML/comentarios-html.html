Comentários html

Os comentários HTML não são exibidos no navegador, mas podem ajudar a documentar seu código-fonte HTML.

Tag de comentário HTML

Você pode adicionar Comentários á sua fonte HTML usando a seguinte sintaxe:

            
<!-- Seu comentário aqui -->

Observe que há um ponto de exclamação (!) na tag start, mas não na tag end tag.

Nota: Os comentários não são exibidos pelo navegador, mas podem ajudar a documentar seu código-fonte HTML.

Adicionar comentários

Com comentários, você pode colocar notificações e lembretes em seu código HTML:

<!--isso é um comentario-->

<!--isso é um comentario-->

Ocultar conteúdo

Os comentários podem ser usados para ocultar o conteúdo.

Isso pode ser útil se você ocultar o conteúdo temporariamente:

E é assim<!--isso é um comentario--> que se esconde algo

<!--isso é um comentario-->

Você também pode ocultar mais de uma linha. Tudo entre o e o ficará oculto da tela.<!-- -->

Oculte uma seção do código HTML:

<!--<h1>isso é um título escondido</h1>-->

<!--<hp>isso é um parágrafo escondido</p>-->

Ocultar conteúdo embutido

Os comentários também podem ser usados para ocultar o conteúdo embutido, como scripts ou estilos:

Ocultar parte de um parágrafo

<!--<h1>isso é um título escondido</h1>-->
