Atributos HTML

Os Atributos HTML fornecem infomações adicionais sobre elementos HTML

Atributos HTML

  • Todos os elementos HTML podem ter atributos
  • Os atributos fornecem infomações adicionais sobre os elementos
  • Os atributos são sempre especificados na tag inicial
  • Os atributos geralmente vêm em pares de nome/valor como: name="value"

O atributo href

A tag define um hyperlink. O atributo especifica a URL da página. O link href vai para: <a>

<a href="https://www.Youtube.com"> Youtube </a>

O atributo src

A tag <img>é usada para incorporar uma imagem em uma página HTML. O atributo srcespecifica o caminho para a imagem a ser exibida.

Exemplo

<img src"HTML/imagens.html/headless horseman.jpeg">

Há duas maneiras de especificar a URL no atributo: src


1. URL absoluto - Links para uma imagem externa hospedada em outro site. Exemplo: src="HTML/imagens.html/headless horseman.jpeg".

Nota: Imagens externas podem estar protegidas por deritos autorais. Se você não tiver permissão para usá-la, você pode estar violando as leis de direitos autorais. E além disso, você não pode controlar imagens externas; pode ser removido repentinamente ou mudou

2. URL relativa - Links para uma immagem hospedada no site. Aqui, o URL não inclui o nome de domínio. Se o URL começar sem uma barra, será relativo á pagina atual. Exemplo: src="headless horseman.jpeg". Se começar com uma barra, ele será relativo ao domínio. Exemplo: src="/imagens.html/headless horseman.jpeg".

Ponto: Quase sempre é melhor usar URLs relativos. Eles não quebrarão se você mudar de domínio.

Os atributod de largura e altura

A tag também deve conter os atributos width e height, que especificam a largura e a altura da imagem (em pixels).

<img src="HTML/imagens.html/headless horseman.jpeg" width="500" height="600">

O atributo alt

O atributo alt requere para a tag <img> um texto alternatico para uma imagem, se a imagem por algum motivo não puder ser exibida. Isso pode ser devido a uma conexão lenta, ou um erro no atributo src.

<img src="HTML/imagens.html/headless horseman.jpeg" alt="cavaleiro sem cabeça">

O atributo de estilo

O atributo style é usado para adicionar estilos a um elemento, como cor, fonte, tamanho, tamanho e muito mais

<p style="color: blueviolet;"> Isto é um parágrafo roxo-azulado.

O atributo lang

Você deve sempre incluir o atributo lang dentro da tag <html>, para declarar o idioma da página da Web. Isso destina-se a auxiliar os mecanismos de pesquisa e navegadores.

O Exemplo a seguir especifica o Português como idioma:

<!DOCTYPE html>

<html lang="pt">

<body>

...

</body>

</html>

Os códigos de país também podem ser adicionados ao código de idioma do atributo lang. Assim, os dois primeiros caracteres define o idioma da página HTML, e os dois últimos caracteres definem o país

O exemplo a seguir define o Português como idioma e o Brasil como país

<!DOCTYPE html>

<html lang="pt-br">

<body>

...

</body>

</html>

O atributo title

title define algumas infomações extras sobre um elemento

O valor do atributo title será exibido como uma dica de ferramenta quando você passar o mouse sobre o elemento

exemplo

<p title="eu sou uma ferramenta">Isto é um parágrafo

Aspas simples ou duplas?

Aspas duplas em torno de valores de atributo são mais comuns em HTML, mas simples também podem ser usadas.

Em algumas cituações, quando o próprio valor do atributo contém aspas duplas, é necessário usar aspas simles:

<p title='John "Shotgun" Nelson'>

Ou vice versa

<p title="John 'Shotgun' Nelson">
