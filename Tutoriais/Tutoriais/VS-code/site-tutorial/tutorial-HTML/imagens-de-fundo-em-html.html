Imagem de fundo

Em HTML, você não define diretamente uma imagem de fundo. Em vez disso, usa CSS para aplicar uma imagem como fundo de um elemento HTML (como <body>,<div><section> etc.).

Sintaxe com CSS

No CSS, usa-se a propriedade background-image.

Exemplo básico

<!DOCTYPE html

<html>

<head>

<style>

body {

background-image: url("caminho/para/sua-imagem.jpg");

background-size: cover;

background-repeat: no-repeat;

background-position: center;

}

</style>

</head>

<body>

Seu conteúdo aqui

</body>

</html>

Explicação das propriedades

  • background-image: Define a imagem de fundo usando a função url() para especificar o caminho da imagem.
  • background-size: Controla o tamanho da imagem de fundo. O valor cover faz com que a imagem cubra todo o elemento.
  • background-repeat: Define se a imagem deve se repetir. O valor no-repeat impede a repetição.
  • background-position: Define a posição inicial da imagem de fundo. O valor center centraliza a imagem.

Considerações finais

Usar imagens de fundo com CSS permite maior controle sobre a aparência e o layout das páginas web. Experimente diferentes propriedades para ver como elas afetam a apresentação da sua imagem de fundo!
