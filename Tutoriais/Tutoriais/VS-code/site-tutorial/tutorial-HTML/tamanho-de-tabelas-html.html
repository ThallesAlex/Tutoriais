Tamanhos de Tabela - HTML

Definindo Largura e Altura

Utilizando Atributos HTML(width e heigth)

Historicamente, era comum definir a largura e altura de tabelas com os atributos width e heigth nas nas próprias tags, como:

HTML

<table width=300 heigth=300

...

</table>

Esses atributos ainda funcionam, mas são considerados depreciados no HTML5. Por isso, recomenda-se utilizar CSS para garantir maior controle, manutenção e compatibilidade futura.

CSS: Definindo Width e Heigth

Para controlar o tamanho de tabelas, colunas e células, utilize as propriedades CSS width e heigth, aplicadas conforme o elemento desejado.

Tabela com largura e altura fixas

CSS

table {

    width: 600px;

    height: 300px;

}

Tabela com largura percentual (responsiva)

CSS

table{

    width100%;

}

Célula ou coluna com largura espacíficas:

CSS

td, th{

    width: 150px

    height: 50px;

}

Essas propriedades podem receber valores em px, %, em, entre outros. Utilizar valores percentuais ou relativos facilita a resposividade da tabela, adaptando-a a diferentes tamanhos de tela

Exemplo prático:

HTML

<table style="width: 90%; height: 250px;">

    <tr>

        <td style="width: 70%; height: 100px;">Coluna Larga<td>

        <td>Coluna Estreita span.tag{</td>}

    <tr>

</table>

Neste exemplo, a tabela ocupa 90% da largura do elemento pai, a primeira célula ocupa 70% da largura da linha e tem uma altura de 100px
