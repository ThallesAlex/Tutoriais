Mapas de imagem HTML

O que é um mapa de imagem?

Um mapa de imagem é uma imagem que contém áreas clicáveis. Cada área pode levar o usuário a uma página diferente, executar um script ou realizar outra ação. É como dividir uma imagem em "zonas interativas"

Como funciona?

Você precisa de três elementos principais:

  1. A imagem com o atributo usemap
  2. A tag <map> que define o mapa
  3. Várias tags <area> que definem as zonas clicáveis

Exemplo básico:

<img src="sua imagem vem aqui" alt="descrição da imagem" usemap="#mapa1">
<map name="mapa1"> <area shape="circle" coords="34,44,270,350" alt="o item da imagem" href="o destino do link"> <area shape="rect" coords="290,172,333,250" alt="o item da imagem" href="o destino do link"> <area shape="poly" coords="337,300,44" alt="o item da imagem" href="o destino do link"> </map>

Explicação dos atributos

  • shape: Define a forma da área clicável. Pode ser "rect" (retângulo), "circle" (círculo) ou "poly" (polígono).
  • coords: Define as coordenadas da área clicável. O formato varia dependendo da forma escolhida.
  • alt: Fornece uma descrição alternativa para a área, útil para acessibilidade.
  • href: Define o destino do link quando a área é clicada.

Aplicações comuns

Mapas de imagem são frequentemente usados em:

  • Mapas geográficos com regiões clicáveis
  • Diagramas interativos
  • Imagens de produtos com partes interativas
  • Jogos baseados em imagens
  • Infográficos com links para mais informações

Nota: Você também pode usar JavaScript com o atributo onclick para executar funções ao clicar em uma ãrea.
