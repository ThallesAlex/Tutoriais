HTML Links

Os links são encontrados em quase todas as páginas da web. Os links permitem que os usuários cliquem de uma página para outra.

Links HTML - Hiperlinks

Links HTML são hiperlinks.

você pode clicar em um link e pular para outro documento.

Quando você move o mouse sobre um link, a seta do mouse se transforma em uma mãozinha.


Nota: Um link nãp precisa ser texto. Um link pode ser uma imagem ou qualquer outro elemento HTML!

Links HTML - Sintaxe

A tag HTML <a> define um hiperlink. Ele tem a seguinte Sintaxe

<a href="url">texto do link

O atributo mais importante do elemento <a> é o atributo href, que indica o destino do link.

O texto do link é a parte que ficará visivel para o leitor.

Clicar no texto do link enviará o leitor para o endereço URL especificado.

Exemplo

Este exemplo mostra como criar um link para W3Schools:

<a href="https://www.w3schools.com/"> Visite W3Schools.com!

Por padrão, os links aparecerão da seguinte forma em todos os navegadores.

  • Links não visitados são azuis e sublinhados
  • Links visitados são roxos e sublinhados
  • Links ativos (clicados) são vermelhos e sublinhados

Ponto: Os links podem, é claro, ser estilizados com CSS, para obter Outro Olhar!


Links HTML - O atributo de destino

Por padrão, a página vinculada será exibida na janela atual do navegador. Para alterar isso, você deve especificar outro destino para o link.

O atributo target especifica onde abrir o documento vinculado.

O atributo target pode ter um dos seguintes valores:

  • _self - Padrão. Abre o documento vinculado na mesma janela ou guia do navegador.
  • _blank - Abre o documento vinculado em uma nova janela ou guia do navegador.
  • _parent - Abre o documento vinculado na janela pai.
  • _top - Abre o documento vinculado na janela inteira do navegador.
  • nome - Abre o documento vinculado em um quadro com o nome especificado.

Exemplo

Use target="_blank" para abrir o documento vinculado em uma nova janela ou guia do navegador:

<a href="https://www.youtube.com/watch?v=W2QeQ9ZufAk&list=RDapK2jCrfnsk&index=4" target="_blank"Youtube

URLs absolutas vs. URLs relativos

Ambos os exemplos acima estão usando um URL absoluto (um endereço da Web completo) no atributo href.

Um link local (um link para uma página dentro do mesmo site) é especificado com um URL relativo (sem a parte "https://www"):

Exemplo

<h2>URL Absoluta</h2>

<p><a href="https://www.youtube.com/watch?v=W2QeQ9ZufAk&list=RDapK2jCrfnsk&index=4">Youtube</a></p>

<h2>URLs Relativas</h2>

<p><a href="html_images.asp"HTML images</a></p>

Links HTML - Use uma imagem como link

Para usar uma imagem como link, basta colocar a tag <img> dentro da tag <a>

Exemplo

<a href="default.asp">
<imgsrc="smiley.gif" alt="HTML Tutorial" style="width:42px;height:42px;"
</a>

Link para um endereço de e-mail

Use mailto: dentro do href para criar um link que abra o programa de e-mail do usuário (para deixe-os enviar um novo e-mail):

<a href="mailto:someone@example.com">enviar e-mail

Botão como um link

Para usar um botão HTML como um link, vovê deve adicionar algum código Java Script.

O Java Script permite que você especifique o que acontece em determinados eventos, como um clique de um botãp:

Exemplo

buttom onclick="document.location='default.asp'">HTML Tutorial </button>

Títulos de links

O atributo title especifica informações extras sobre um elemento. As informações são mostradas com mais frequência como um texto de dica de ferramenta quando o mouse se move sobre o elemento.

Exemplo

<a href"https://www.youtube.com/watch?v=NMRhx71bGo4&list=RDNMRhx71bGo4&start_radio=1" title="Um vídeo do Youtube"Let It Happen</a>
