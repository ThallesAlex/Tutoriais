Cores HTML - HSL a HSLA

HSL significa matiz (Hue), saturação e luminosidade.

Os valores de cor HSLA são uma extensão do HSL com um canal alfa (opacidade).

Valores de cor HSL

Em HTML, uma cor pode ser especificada usando matiz (Hue), saturação e luminosidade (HSL) no formulário:

HSL (matiz(Hue), saturação, luminosidade)

Matiz (Hue), é um grau na roda das cores de 0 a 360. 0 é vermelho, 120 é verde e 240 é azul

A saturação é um valor percentual. 0% significa um tom de cinza e 100% é a cor completa.

A saturação também é um valor percentual, 0% é preto e 100% é branco.

Exemplo

hsl(0, 100%, 50%)

hsl(240, 100%, 50%)

hsl(147, 50%, 47%)

hsl(39, 50%, 10%)

hsl(90, 27%, 40%)

hsl(89, 70%, 46%)

Saturação

A saturação pode ser descrita como a intensidade de uma cor.

100% é cor pura, sem tons de cinza.

50% é 50% cinza, mas você ainda pode ver a cor.

0 é componente cinza; Você não pode ver mais cor.

hsl(0, 100%, 50%)

hsl(0, 60%, 50%)

hsl(0, 0%, 50%)

Leveza

A luminosidade de uma cor pode ser descrita como a quantidade de luz que você deseja dar à cor, onde 0% significa nenhuma luz (preto), 50% significa 50% claro (nem escuro nem claro), e 100% significa luminosidade total (branco).

hsl(0, 100%, 0%)

hsl(0, 100%, 50%)

hsl(0, 100%, 75%)

hsl(0, 100%, 100%)

Tons de cinza

Tons de cinza são frequentemente definidos definindo o matiz e a saturação como 0, e Ajustando a luminosidade de 0% a 100% para obter tons mais escuros/claros:

hsl(0, 0%, 20%)

hsl(0, 0%, 90%)

hsl(0, 0%, 50%)

Valores de cor HSLA

Os valores de cor HSLA são uma extensão dos valores de cor HSL, com um canal alfa - que especifica a opacidade de uma cor.

Um valor de cor HSLA é especificado com:

hsla (matiz (Hue), saturação, luminosidade, alfa)

O parâmetro alfa é um número entre 0,0 (totalmente transparente) e 1,0 (nada transparente):

Exemplo

hsla(60, 100%, 64%, 0)

hsla(60, 100%, 64%, 0.4)

hsla(60, 100%, 64%, 0.6)

hsla(60, 100%, 64%, 1)
