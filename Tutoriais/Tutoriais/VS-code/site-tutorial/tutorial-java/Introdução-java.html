História e Evolução do Java

Java foi criado por James Gosling e sua equipe na Sun MicroSystens como uma linguagem para dispositivos eletrônicos inteligentes. Lançada publicamente em 1995. sua principal inovação foi a independência de plataforma, graças à JVM (Java Virtual Machine), que permite que o código compilado (bytecode) rode em qualquer sistema com JVM instalada. Em 2010, a Oracle adiquiriu a Sun, continuando o desenvolvimento em performance e sintaxe.

Características Principais do Java

Java se destaca por várias features que a tornam atraente para desenvolvedores:

  • Indepanderncia de Plataforma: "Write Once, Run Anywhere" (WORA) - o código é compilado para bytecode e executado pela JVM em Windows, macOS, Linux, etc:
  • Orientação a Objetos:: Suporta conceitos como classes, herança, polimorfismo e encapsulamento para código modular e reutilizável
  • Robusta e Segura: Gerenciamento automático de memória(garbage collector) e tratamento de execuções evitam erros comuns.
  • Multithreading: Permite execução concorrente de tarefas, ideal para aplicações eficientes
  • Rica em Bibliotecas: APIs padrão e frameworks como Spring para desenvolvimento Web
  • Comunidade e Suporte: Milhões de desenvolvedores e recursos gratuitos disponíveis.

Comfigurando o Ambiente de Desenvolvimento

Para começar, siga estes passos detalhados:

    Instale o JDK:
    • Acesse o site da Oracle (apenas uma imagen grande) e baixe a versão mais recente (ex: Java 21 ou superior) para seu SO.
    • No Windows: Execute o instalador e siga as instruções.
    • No macOS/Linux: Use gerenciadores de pacotes ou instalador

    Configure Variáveis de Ambiente:
    • Defina JAVA_HOME para o diretório de instalação (ex: C:\Program Files\Java\jdk-21 no Windows).
    • Adicione o bin ao PATH
    • Verifique com java - version no terminal.

    Escolha um IDE:
    • Recomendado: Intelij IDEA Community (gratuito)
    • Alternativas: Eclipse ou VS Code com extensões Java.
    • No Intelij, configure o JDK em Project Structure > SDKs

    Escreva Seu Primeiro Programa:

    Crie um arquivo ZaWarudo.java:

    Java

    public class ZaWarudo {
    public static void main(String[] args ) {
    System.out.println("Za Wardo!");
    }
    }

    Compile com javac ZaWarudo.java e execute como java ZaWarudo.

Sintaxe Básica e Tipos de Dados

A sintaxe de Java é semelhante a C++, sensível a MAIÚSCULAS/minúsculas, e requer ponto-e-vírgula ao final de statements.

  • Método Main: Ponto de entrada: public static void main(String[] args)...
  • Impressão: System.out.println("texto");
  • Variáveis: Declare com tipo, ex: int numero = 10
  • Comentários: // Linha única ou /*Múltiplas linhas*/

Tipos de dados primitivos em Java:

Tipo Descrição Tamanho Exemplo
byte Inteiro pequeno 1 byte byte b = 100;
short Inteiro curto 2 bytes shorts s = 1000;
int Inteiro padrão 4 bytes int i = 10000;
long Inteiro longo 8 bytes long l = 100000L;
float Ponto flutuante simples 4 bytes float f = 3.14f;
double Ponto flutuante duplo 8 bytes double d = 3.14;
boolean Verdadeiro/Falso 1 bit boolean bool = true;
char Caractere único 2 bytes char c = 'a';

Objetos como String são não-primitovos: String texto = "Bury The Light Deep Within";
