Tradução do conteúdo do terminal

Este conteúdo é uma tradução adaptada da documentação exibida via comando info no terminal Linux. O texto original está disponível sob a licença GNU Free Documentation License, conforme publicado pela Free Software Foundation.

Esta tradução tem fins exclusivamente educacionais e não substitui a versão oficial. Créditos ao projeto original e aos autores da documentação.

eu resumi algumas coisas, e em outras coisas usei IA para ter um detalhamento mais legível para certas pessoas
  • Arquivamento
  • Básicos

Introdução

Arquivo: dir (diretório), Node: topo, Esse é o topo da árvore INFO.

Esse é o menu principal do Info.
Alguns comandos de informações úteis:

  • 'q' sai;
  • 'H' lista todos os comandos do Info;
  • 'h' começa o tutorial Info;
  • 'mTextinfo RET' visita o manual Textinfo, etc.
