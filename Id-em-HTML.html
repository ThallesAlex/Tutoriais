Ids - HTML

O atributo id em HTML serve para identificar de forma única um elemento dentro de uma página. Ele é essencial para aplicar estilos com CSS, manipular elementos com JavaScript e criar âncoras para navegação

O que é o atributo id?

  • Definição: o id é um atributo global que atribui um identificador único a um elemento HTML
  • Unicidade : Cada id deve ser exclusivo dentro do documento - não pode haver dois elementos com o mesmo id

Sintaxe

HTML


<div id="meuElemento">Conteúdo aqui</div>

  • O valor do id deve ser uma linha sem espaços
  • Evite usar caracteres espaciais ou iniciar com números

Usos comuns id

  • Estilização com CSS

    Você pode aplicar estilos específicos a um elemento com id:


    <style>
       #meuElemento {
          color: blue;
    background-color: yellow;
    }
    </style>

  • Manipulação com JavaScript

    Permite selecionar e modificar elementos:

    HTML


    <script>
    document.geiElementById("meuElemento").innerText = "Texto alterado!";
    <script>

  • Navegação por âncoras

    Crie links que levam diretamente a uma seção da página:

    HTML

    <a href="#meuElemento"Ir para a seção</a>

    <div id="maeuElementos"Seção de destino</div>;

Boas Práticas

  • Use id apenas quando realmente precisar identificar um elemento de forma única
  • Prefira class para estilos em múltiplos elementos.
  • Mantenha nomes de id descritivos e consistentes.
