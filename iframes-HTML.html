Iframe em HTML

Um iframe(inile frame) é um elemento HTML que incorpora uma outra página HTML dentro da página atual. criando um contexto de navegação aninhado com histórico e documentos próprios

Sintaxe básica

  • Elemento <iframe>
  • Atributo essencial: src(URL da Página a ser incorporado)

Exemplo mínimo

HTML


<iframe src="https://exemplo.com" title="Conteúdo incorporado" width="600" height="400"></iframe>

Atributos mais úteis

  • src - URL do conteúdo a ser exibudo.
  • title - Descrição acessível do conteúdo (Iportante para acessibilidade)
  • width e height - Dimensões do iframe (em px ou %)
  • loading - Pode receber lazy para carregamento sob demanda.
  • sandbox - Restringe capacidades do conteúdo incorporado (várias flags possíveis, ex: allow-scripts, allow-same-origin). Use para reduzir riscos de segurança
  • allow - dá permissões como camera, microphone, fullscreen.
  • referrerpolicy - Controla o envio do referer para o conteúdo embutido

Segurança e privaxidade (princípios rápidos)

  • Conteúdo de terceiros pode executar scripts; por isso use sandbox para limitar ações (evitar navegação, execução de scripts, envio de formulários etc.). Atributos de sandbox permitem flexibilizar retrições quando necessário
  • Políticas de servidor alvo podem impedir incorporação (ex: cabeçalhos como X-Frame-Options ou políticas de Content Security Policy), por isso nem toda URL pode ser mostrada em iframe
  • Evite injetar URLs controladas por usuários sem validação/normalização para mitigar riscos de clickjacking e open-redirectis.

Boas práticas

  • Sempre inclua title para acessibilidade.
  • Prefira loading="lazy" para iframes não críticos (melhora desenpenho de carregamento)
  • Se o conteúdo for de origem não externa não confiável, conbine sandbox com políticas de segurança mais rígidas.
  • Para conteúdo que precisa interagir com a página mãe, use mensagens postMessage com origem verificada (não confiar em dados sem validação).

Exemplo avançado com sandbox e permissões

HTML


<iframe
   src="https://player.exemplo.com/video/473"
   title="Player de video"
   width="800"
   height="450"
   loading="lazy"
   sandbox="allow-scripts allow-same-origin"
   allow="accelerometer; autoplay; encrypted-media; picture-in-picture; fullscreen"
</iframe>
